List of commands

  • git init - Creates a new local repository
  • git add - Indexes the specified file for later commit
  • git commit - Commits indexed changes and saves them to version history
  • git status - Lists all new or changed files that need to be committed
  • git diff - Shows differences by changes made in files not yet indexed
  • git reset - Cancels the indexing of the specified file, while preserving its contents
    soft - with tracking
    mixed - without tracking
    hard - deleted
